## About <a id="about"></a>
This website was created for VII INK, a luxury tattoo studio in London, Ontario that blends Eastern and Western tattoo traditions.

## Skills ✨ <a id="skills"></a>

### Languages: 🛠️
JavaScript, HTML5, CSS3, PHP, SQL

### Design Tools: 🎨
Adobe Illustrator, Adobe After Effects, Figma

### Development Tools: 
VS Code, GitHub, SASS, Laravel, MySQL

### Frameworks & Libraries:
Vue.js, GSAP, Fetch API

## Features 📋 <a id="features"></a>
  ⚡️ Fully Responsive Design
  ⚡️ Dynamic Artist Gallery (Fetch API)
  ⚡️ Portfolio Filtering System
  ⚡️ PHP/AJAX Contact Form
  ⚡️ User Authentication
  ⚡️ Image Upload Functionality
  ⚡️ Laravel Backend API
  ⚡️ MySQL Database Integration
  ⚡️ GSAP Scroll Animations

## Installation 📦 <a id="installation"></a>
To run the portfolio locally:

- Clone this repository
- Navigate into the project directory:
- Open the index.html file in your browser, or if using a local server, start the server and navigate to localhost to view.
